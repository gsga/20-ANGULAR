Si se pone texto o html suelto en index.html, se mostrará como html sin más. Notar que la interpolación no funciona, sino que aparece textualmente.

Se puede poner en index.html otros componentes además de app-root, pero hay que agregarlos a la sección bootstrap, como se explica en la siguiente respuesta de StackOverflow:
