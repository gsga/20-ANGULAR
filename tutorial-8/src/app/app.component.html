Ejemplo de interpolación. Entre las dobles llaves tenemos la variable titulo. Esta variable es declarada en app.component.ts, donde también se le asigna un valor. La información fluye desde el ts al html.

Interpolación de cuentas

Ejemplo de interpolación de cuentas. Entre las dobles llaves tenemos una expresión. Esta expresión es evaluada y el resultado se interpola.

7 * 8 = {{7 * 8}}

Ejemplo de interpolación de función. Entre las dobles llaves tenemos un llamado a una función. La función está definida en el archivo ts, no en este template. La función retorna un valor, y ese valor se interpola entre las llaves dobles.

suma(2, 2) = {{suma(2, 2)}}
